# AIDA

### Requerimientos
- Se da por asumido que se tiene creada la base de datos `aida_db` con el schema `aida`
- node version `>= 22.18.0`

### How to run
- Se da por asumido que se tiene creada la base de datos `aida_db`
- `npm i` para instalar dependencias.
- Setupear variables de entorno en `.env`
- `npm start` para correr el código.

Para variables de entorno, crear un archivo `.env`, usar `.env.example` como guía
